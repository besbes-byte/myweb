# Thread Note

한 문장 메모 + 리플 구조 + 사진 첨부를 지원하는 개인용 스레드 메모 앱 프로토타입입니다.

## 기능
- 루트 메모 작성
- 리플(대댓글) 구조
- 사진 첨부
- 검색
- 고정 메모
- 내보내기 / 불러오기(JSON)
- 오프라인 캐시(PWA 기본 지원)

## 실행 방법
### 가장 쉬운 방법
브라우저에서 `index.html`을 열면 됩니다.

### 로컬 서버 권장
일부 브라우저에서 서비스워커/PWA 기능을 제대로 보려면 로컬 서버가 좋습니다.
예:
```bash
python3 -m http.server 8080
```
그 뒤 브라우저에서 `http://localhost:8080` 접속.

## 주의
- 첨부 사진은 브라우저 저장소(IndexedDB)에 저장됩니다.
- 기기나 브라우저를 바꾸면 자동 동기화되지 않으니, 필요할 때 JSON 내보내기로 백업하세요.
- 완전한 iOS/Android 앱으로 배포하려면 이후 Capacitor 또는 React Native로 감싸면 됩니다.
